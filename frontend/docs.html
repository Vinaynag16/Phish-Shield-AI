    Project Overview

    PhishShield AI is a proactive defense system designed to identify social engineering threats in real-time using Machine Learning.

    System Architecture

    The system utilizes a FastAPI backend to bridge the gap between our Scikit-learn models and the Frontend interface.

    Machine Learning Logic

    We use a Random Forest Classifier for URL analysis, achieving high accuracy by evaluating features like domain length, special characters, and HTTPS status.
